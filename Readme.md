# MM ImageVault
MM ImageVault is a Django based web application that allows users to upload, download and manage images temporarily while preserving their original quality and file format.

## Features
- Multiple image upload support 
- Supports various image formats(JPG, PNG, WEBP, GIF, BMP, TIFF)
- Original image quality preservation
- Original file format retention
- Image download functionality
- Image deletion functionality
- Automatic file expiration after a specified period
- User-friendly Interface

## Technology Used
- Python
- Django
- HTML
- CSS
- SQLite
- Pillow

## Project Objective
The object of MM ImageVault is to provide a temporary image storage solution that enabkes users to upload and download images without any loss in quality or changes in file format. Uploaded files are automatically removed after a predefined period to optimize storage management.

## Workflow
1. Upload one or more image file.
2. Store images temporarily on the server.
3. Download images in their original format and quality.
4. Delete images manualyy if required.
5. Automatically removed expired files after the specified duration.
