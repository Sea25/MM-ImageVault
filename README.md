# MM-ImageVault
MM ImageVault is Djanfo based web application for temporary image upload, download and management while preserving original image quality and file format.

## Features
- Multiple image upload support
- Supports various image formats (JPG, PNG, WEBPGIF, BMP, TIFF)
- Original image quality preservation
- Original file format retention
- Image download functionality
- Image deletion functionality
- Automatic file expiration after a specified time period.
- User-friendly interface

## Technologies Used
- Python 
- Django
- HTML
- CSS
- SQLite
- Pillow

## Project Obejective
The objective of MM ImageVault is to provide a temporary image storage solution that enables users to upload and download image  without any loss in quality or changes in file format. Uploaded files are automatically removed after a predefined period to optimize storage management.

## Workflow
1. Upload 1 or more image files.
2. Store images temporarily on the server.
3. Download images in their original format and quality.
4. Delete images manually if required.
5. Automatically remove expired files after a specified duration.
